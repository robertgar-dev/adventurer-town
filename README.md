# Adventurer Town

Sprint 01 foundation implementation.

This codebase establishes the Flutter/Riverpod shell, pure Dart simulation
domain, local persistence boundary, analytics wrapper, and deterministic
headless tick skeleton for Adventurer Town.

The current build intentionally excludes Town View, Building Detail UI,
offline progression, monetization, combat, inventory, crafting, equipment,
classes, quests, staffing, trade routes, and specialization.
